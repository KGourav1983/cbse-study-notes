# Mock Paper — Rational Numbers

## Time Allowed: 1 hour
## Maximum Marks: 25
## General Instructions
- This paper covers Class 8 Mathematics, Chapter 1 — Rational Numbers.
- All questions are compulsory.
- Section A carries 1 mark per question, Section B carries 2 marks per question, and Section C carries 3 marks per question.
- Show all working for Section B and C questions — no marks are awarded for a final answer alone.

## Section A — 1 Mark Questions
1. Define a rational number.
2. Write the additive identity for rational numbers.
3. What is the reciprocal of −7/9?
4. State whether true or false: "0 has a multiplicative inverse."
5. Is subtraction of rational numbers commutative?

## Section B — 2 Mark Questions
6. Find the additive inverse of 5/(−8).
7. Using the averaging method, find one rational number between 2/5 and 3/5.
8. Verify: 3/4 + (−2/3) = (−2/3) + 3/4.
9. Simplify: (−4/5) × (10/(−9)).
10. Name the property illustrated by: 2/3 × (3/4 × 5/6) = (2/3 × 3/4) × 5/6.

## Section C — 3 Mark Questions
11. Verify the distributive property a × (b + c) = a × b + a × c for a = 2/5, b = 1/3, c = 1/2.
12. Find three rational numbers between 1/5 and 2/5, showing your method.
13. Using appropriate properties, simplify: 2/3 × (−4/7) − 1/3 × 6/7 + 2/3.

## Answer Key
1. A number expressible in the form p/q, where p and q are integers and q ≠ 0.
2. 0 (zero) — since a + 0 = a for every rational number a.
3. −9/7.
4. False — 0 has no multiplicative inverse.
5. No — subtraction of rational numbers is not commutative.
6. 5/8 (since 5/(−8) = −5/8, its additive inverse is 5/8).
7. (2/5 + 3/5) ÷ 2 = 1 ÷ 2 = 1/2.
8. Both sides equal 1/12, confirming the commutative property of addition.
9. (−4/5) × (10/(−9)) = 40/45 = 8/9.
10. The associative property of multiplication.
11. Left side: 2/5 × (1/3 + 1/2) = 2/5 × 5/6 = 1/3. Right side: (2/5 × 1/3) + (2/5 × 1/2) = 2/15 + 1/5 = 2/15 + 3/15 = 5/15 = 1/3. Both sides equal 1/3, verifying the property.
12. Using common denominator 20: 1/5 = 4/20 and 2/5 = 8/20, so 5/20, 6/20, and 7/20 (i.e. 1/4, 3/10, 7/20) all lie between them.
13. Group the terms with 2/3 using the distributive property: 2/3 × (−4/7 − 6/7... ) — regrouping gives 2/3 × (−4/7) − 1/3 × 6/7 + 2/3 = −8/21 − 2/7 + 2/3 = −8/21 − 6/21 + 14/21 = 0/21 = 0.
